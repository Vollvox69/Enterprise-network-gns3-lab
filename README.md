Enterprise Multi-Site Network Lab (GNS3)

This project simulates a multi-branch enterprise network using GNS3.

Technologies:
- RIP
- EIGRP
- Autonomous System configuration
- Administrative Distance tuning
- Load Balancing
- Unload Balancing
- EIGRP Summarization 
- Subnetting (/27, /30)

The lab demonstrates routing between multiple city routers with optimized traffic distribution.
